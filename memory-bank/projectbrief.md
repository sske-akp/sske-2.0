# projectbrief.md

This document defines the core requirements and goals for the project.  
It is the source of truth for project scope and shapes all other memory bank files.

**Project Summary:**  
A modular business management application for handling sales, purchases, customers, and stock.  
**Objectives:**

- Enable interactive sales and purchase invoice creation
- Manage product inventory, pricing, and categories
- Provide a modern, responsive UI for business operations
- Support extensibility via modular components and hooks
