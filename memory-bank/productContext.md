# productContext.md

This document explains why the project exists, the problems it solves, how it should work, and user experience goals.

**Product Context:**  
Designed for small to medium businesses to manage sales, purchases, inventory, and customer data in one place.

**User Needs:**

- Quickly create and edit sales/purchase invoices
- Track product inventory and pricing
- Manage customer and supplier information
- Access a unified, intuitive dashboard

**Experience Objectives:**

- Fast, responsive UI
- Minimal manual data entry
- Clear, actionable feedback for all operations
- Modular, extensible workflows
