# techContext.md

This document details the technologies used, development setup, technical constraints, dependencies, and tool usage patterns.

**Technologies Used:**

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Radix UI (various components)
- TanStack Table & Query
- Sonner (notifications)
- Lucide React (icons)
- Custom modular UI components

**Development Setup:**

- Node.js (see package.json for version)
- Run `npm install` to install dependencies
- Start dev server: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`

**Dependencies:**

- See package.json for full list of dependencies and devDependencies

**Tool Usage Patterns:**

- Modular hooks for data fetching and state
- File-based routing for feature modules
- Tailwind for utility-first styling
